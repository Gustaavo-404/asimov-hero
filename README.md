# Asimov Hero Section

Este projeto foi desenvolvido como parte de um **teste técnico para processo seletivo**, com foco no desenvolvimento de uma seção "hero" moderna para uma landing page de curso

## 🚀 Tecnologias utilizadas

- Next.js
- React

## 🎯 Objetivo

Desenvolver uma seção hero com:

- Layout moderno e responsivo
- Uso de assets visuais
- Organização de componentes
- Código limpo e reutilizável

## 🧩 Estrutura

- `app/` → estrutura da aplicação
- `components/` → componente principal (`AsimovHero`)
- `public/` → gifs e imagens utilizadas

## 🌐 Deploy

Publicado via Vercel.

Observação: Projeto focado em frontend
